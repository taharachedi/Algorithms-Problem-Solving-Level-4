# 📅 Problem 6: Month Days Calculator

This simple C++ program calculates the number of days in a given month of a specified year, with full support for leap years! 🌟

## 🛠 Features

- 🔢 **Flexible Input**: Enter any year and month.
- 🗓 **Leap Year Awareness**: Automatically adjusts for leap years in February.
- ⚙️ **Validated Month Entry**: Ensures the month input is between 1 and 12.

## 🚀 How to Use

1. Run the program. 🏃‍♂️
2. Enter a **year** when prompted. 📆
3. Enter a **month** (1-12) when prompted. 📅
4. See the number of days displayed instantly! ✅
